## Selected projects

### [Venuefinder](https://github.com/MKmuneebkhalid/venuefinder)

AI-assisted venue discovery built with Django, Django REST Framework, React, PostgreSQL/pgvector, Redis, and Celery. It combines vector retrieval with structured LLM reranking and runs heavier search work asynchronously.

### [ERP-Lite](https://github.com/MKmuneebkhalid/erp-lite)

A lightweight task and resource management platform using React, TypeScript, Spring Boot, PostgreSQL, and MongoDB, with Docker-based local setup and a CI pipeline.

### [Meta AI Analyst](https://github.com/MKmuneebkhalid/meta-ai)

A read-only analytics service for Meta advertising data, including scheduled snapshots, natural-language analysis, diagnostics, and evidence-based reporting.

### [OpenHome Abilities](https://github.com/MKmuneebkhalid/abilities)

A collection and framework for modular voice-AI capabilities written in Python, with reusable templates for API integrations, multi-turn interactions, and custom agent behavior.
